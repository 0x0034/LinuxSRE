## SMTP

> Simple Mail Transfer Protocol，简单邮件传送协议

- 邮件发送协议
- 传输路由的功能
- C/S架构

- 客户 -> A邮件服务器(邮件中继) -> B邮件服务器(邮件中继) -> 目标邮件服务器

### SMTP 如何工作

- Servers: smtpd(sendmail)
- Client: smtp
